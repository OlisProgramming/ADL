Mathematical Expressions

This token type's value is an Array of tokens - the tokens that make up the expression (e.g. 42i | + | 138i).

The MathExpression class then, at tokenization time, uses the shunting-yard algorithm to convert the infix token order to postfix token order (see the Wikipedia page on it) because it is more efficient to parse postfix tokens rather than infix tokens.
