What are Tokens?

Tokens are the building blocks of ADL. Each individual symbol, name or mathematical expression is classed as a token. Look at the example ADL code below and how it is separated into tokens (the verical bar | is used to denote token separators).

int addition = <42i + 138i> is tokenized to

int | addition | = | < | 42i | + | 138i | >

However, this is not entirely accurate. Because mathematical expressions can be quite complex, they have their own type - MATH_EXPRESSION. Check out the MathExpression page for more info.
