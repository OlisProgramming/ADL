List of Token Types

This is a comprehensive list of all of ADL's token types (last updated version 0.1.2).

Token Type Name in code Information
Name STR_NAME Any valid character string to make a name (all characters must be letters, upper or lower case, with no other characters; it must be lowerCamelCase), e.g. exampleName.
String Literal STR_LITERAL Any set of characters enclosed between "double quote marks", and thus you must exclude the double quote mark itself in all strings.
Integer Value VAL_INT Any integer value. Must end in i, standing for integer.
Text Keyword KWD_TEXT The keyword text.
Int Keyword KWD_INT The keyword int.
Mathematical Expression MATH_EXPRESSION Any mathematical expression, enclosed within <angle brackets>.
Print Function FNC_PRINT The stdlib (standard library) function print. Can print any variable type overriding toString.
Delete Function FNC_DELETE The stdlib (standard library) function delete. Deletes variables (there is no ADL garbage collector).
Plus Operator OPR_PLUS The addition operator.
Assignment/Equals Operator OPR_EQUALS The variable assignment operator. NOT used as an equality operator.
End Of File EOF The End Of File marker. Automatically generated when the file has reached the end by the tokenizer.
