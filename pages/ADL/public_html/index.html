Code With Simplicity.

Welcome to ADL! ADL (the App Development Language) is in a very early alpha stage at the moment. You can perform some limited simple tasks, for example evaluating simple integer mathematical expressions or assigning and printing variables. However, some of the core features of ADL are already in place and thus I feel it is necessary to create a formal language specification. This means that in the future, if I happen to forget what parts of the code do (or, indeed, someone else wants to help me create this language) documentation is easily accessible.

Links to many of the core features of ADL are below. Click the links to get started!
